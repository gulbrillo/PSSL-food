LunchPad · Discord Bot Setup

Connecting lunchpad.darkcosmos.org to the PSSL Shenanigans Discord server

What it does

  • 48 h before each RSVP deadline: posts a reminder in your lab channel (meeting, caterer, deadline, response count).
  • 24 h before: optionally DMs every linked member who hasn't responded — off by default, admins toggle it under Admin → Settings on the website.
  • Slash commands: /rsvp yes|no [request] and /meeting — RSVP without opening the website.
  • Members link their Discord once via Profile → Connect Discord on the website.

Where does it run?

On your own server — it is the bot container of the existing docker-compose stack. No extra hosting needed. It opens an outbound connection to Discord; no inbound ports, no public URL. It talks to the web app internally over the Docker network.

1 · Create the Discord application

  1. Go to discord.com/developers/applications → New Application, name it LunchPad.
  2. OAuth2 tab → copy Client ID; click Reset Secret → copy Client Secret.
  3. Same tab, under Redirects, add:
    https://lunchpad.darkcosmos.org/auth/discord/callback
  4. Bot tab → Reset Token → copy the Bot Token. (No privileged intents needed.)

2 · Invite the bot to PSSL Shenanigans

  1. OAuth2 → URL Generator: check scopes bot + applications.commands.
  2. Bot permissions: Send Messages.
  3. Open the generated URL, pick PSSL Shenanigans, authorize. (You need Manage Server rights there.)

3 · Copy the reminder channel's ID

  1. Discord → Settings → Advanced → Developer Mode: ON.
  2. In PSSL Shenanigans, right-click the channel where reminders should be posted (e.g. #lab-meetings) → Copy Channel ID.
The server ID is already known — PSSL Shenanigans = 1063560400332927076 (pre-filled below).

4 · Enable it on the server (after the fact)

Skipped Discord during install.sh? Just fill in the five values in .env — no reinstall needed. 
cd PSSL-food
nano .env
DISCORD_CLIENT_ID=…from step 1.2
DISCORD_CLIENT_SECRET=…from step 1.2
DISCORD_BOT_TOKEN=…from step 1.4
DISCORD_GUILD_ID=1063560400332927076
DISCORD_CHANNEL_ID=…channel ID, step 3
docker compose up -d   # recreates the bot with the new config
docker compose logs -f bot   # expect: "Logged in as …",
                             #         "Slash commands registered"

5 · Tell the lab

Everyone opens lunchpad.darkcosmos.org → Profile → Connect Discord (one click). Only linked members can get DM reminders and use /rsvp — unlinked members still see the channel reminders.

Tuning & troubleshooting

CHANNEL_REMINDER_HOURShours before deadline for the channel post (default 48)
DM_REMINDER_HOURShours before deadline for personal DMs (default 24)
  • Bot container restart-loops until the token is set — that's expected, harmless.
  • No DMs received? The member's privacy settings must allow DMs from server members.
  • Slash commands appear seconds after startup (registered per-server, not global).
